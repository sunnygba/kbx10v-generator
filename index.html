📖 使用說明書

🔗 KING BLADE X10 V 官方說明書 (日文)

1. 基本操作 (預設模式：15 色循環)

  • 電源開啟： 按住 A 鍵約 0.5 秒後放開。（若從關機狀態下長按，會從第 1 色開始亮起）。
  • 電源關閉： 長按 A 鍵約 1.5 秒。
  • 顏色切換： 按 A 鍵依照顏色順序往後切換；按 B 鍵依照顏色順序往前切換。
  • 預設顏色順序： 紅 → 藍 → 白 → 橘 → 綠 → 紫 → 粉 → 黃 → 淺綠 → 淺藍 → 淺粉 → 紫羅蘭 → 青檸綠 → 土耳其藍 → 熱情粉。

2. 記憶功能 (自訂顏色順序與數量)

您可以將自訂的顏色順序儲存在「記憶 1」或「記憶 2」中（最多可設定 30 種顏色）。

  • 設定記憶 1： 關機狀態下，長按 B 鍵 3 秒（紅燈慢閃）。按 A 鍵更換顏色，按 B 鍵選定顏色（選定後閃爍會變快）。設定完成後，長按 B 鍵關機即可儲存。
  • 設定記憶 2： 關機狀態下，同時長按 A 鍵與 B 鍵 3 秒（藍燈慢閃）。同樣按 A 鍵更換、B 鍵選定。設定完成後，長按 B 鍵關機即可儲存。
  • 選擇記憶模式： 關機狀態下，長按 B 鍵 3 秒（紅燈閃爍）。按 C 鍵選擇模式：紅燈為記憶 1、藍燈為記憶 2、白燈為預設模式。選定後長按 B 鍵關機即可完成切換。
  • 清除記憶： 進入選擇記憶模式（B 鍵長按 3 秒，紅燈閃）。按 C 鍵選擇要清除的記憶（紅為 1，藍為 2）。同時長按 A 與 B 鍵即可清除（預設模式無法清除）。

3. 智慧型手機顏色自訂 (App 連接)

  • 連線步驟： 開啟手機藍牙與定位功能。開啟手燈，同時長按 A 鍵與 B 鍵，進入藍燈慢閃狀態（調整模式）。開啟 App 點擊「Pairing」，選擇「KBX5」，閃爍變快即代表連線成功。
  • App 設定： 使用「+Create Blade」新增發光模式，使用「+Create Color」新增顏色。調整 RGBW 數值後點擊「Select」。設定完成後點擊「Install」，選擇存入記憶 A 或 B。手燈全亮即代表安裝完成。

4. 快速跳轉功能 (Hot Button)

  • 設定： 將手燈切換到你最喜歡的顏色（推し色），長按 C 鍵約 3 秒，燈光閃爍一次即設定完成。
  • 使用： 隨時按下 C 鍵即可瞬間跳轉至該顏色。
  • 返回： 跳轉後，按下 A 或 B 鍵即可回到跳轉前的顏色。

(註：若您在使用過程中遇到困難，請確認電池電量是否充足，並確保手機藍牙功能正常。)

🔗 KING BLADE X10 V Official Manual (JP)

1. Basic Operations (Default Mode: 15-Color Change)

  • Power ON: Press and hold switch A for approx. 0.5 seconds, then release.
  • Power OFF: Press and hold switch A for approx. 1.5 seconds.
  • Color Change: Press switch A to advance the color order. Press switch B to reverse the color order.
  • Default Color Order: 1.RED → 2.BLUE → 3.WHITE → 4.ORANGE → 5.GREEN → 6.PURPLE → 7.PINK → 8.YELLOW → 9.LIGHT GREEN → 10.LIGHT BLUE → 11.LIGHT PINK → 12.VIOLET → 13.LIME → 14.TURQUOISE → 15.HOT PINK.

2. Memory Function (Customize Color Order/Count)

You can save custom color sequences to "Memory 1" or "Memory 2" (up to 30 colors per memory).

  • Setting Memory 1: With the device OFF, press and hold switch B for 3 seconds (RED blinks slowly). Press A to change colors, press B to select a color. Once finished, press and hold switch B until it turns OFF to save.
  • Setting Memory 2: With the device OFF, press and hold switches A and B simultaneously for 3 seconds (BLUE blinks slowly). Use the same selection method. Press and hold B until OFF to save.
  • Selecting a Memory Pattern: With the device OFF, press and hold switch B for 3 seconds (RED blinks). Press switch C to choose your pattern: RED (Memory 1), BLUE (Memory 2), or WHITE (Default). Press and hold switch B to confirm and turn OFF.
  • Resetting Memory: Enter memory selection mode (Hold B for 3 sec, RED blinks). Press C to select memory (RED for 1, BLUE for 2). Press and hold A and B simultaneously to clear.

3. Color Tuning Function (App Connection)

  • Connection: Turn on Bluetooth and Location Services. Turn the light ON, then press and hold A and B simultaneously until BLUE blinks slowly. Open the app, tap "Pairing," select "KBX5".
  • App Usage: Select your model. Use "+Create Blade" to create a new list and "+Create Color" to add colors. Adjust RGBW values. Tap "Install" and select Memory A or B.

4. Hot Button Function

  • Setting: Light up your favorite color ("Oshi-iro"), then press and hold switch C for approx. 3 seconds. The light will blink once.
  • Usage: While any color is lit, press switch C to instantly jump to your favorite color.
  • Return: While your favorite color is lit, press switch A or B to return to the previous color.

🔗 KING BLADE X10 V 公式マニュアル

1. 基本操作 (デフォルトモード：15色チェンジ)

  • 電源ON： Aボタンを約0.5秒間長押しして離す。（OFFから長押しすると1色目から点灯）。
  • 電源OFF： Aボタンを約1.5秒間長押し。
  • カラーチェンジ： Aボタンで順送り、Bボタンで逆送り。
  • デフォルトカラー順： 1.RED → 2.BLUE → 3.WHITE → 4.ORANGE → 5.GREEN → 6.PURPLE → 7.PINK → 8.YELLOW → 9.LIGHT GREEN → 10.LIGHT BLUE → 11.LIGHT PINK → 12.VIOLET → 13.LIME → 14.TURQUOISE → 15.HOT PINK.

2. メモリー機能 (カスタムカラー順と数)

自作のカラー順序を「メモリー1」または「メモリー2」に保存できます（最大30色）。

  • メモリー1作成： OFFの状態でBボタンを3秒長押し(赤点滅)。Aで色変更、Bで決定。Bボタン長押しで電源OFFにして保存。
  • メモリー2作成： OFFの状態でAとBボタンを同時3秒長押し(青点滅)。同様に選択し、B長押しで保存。
  • モード選択： OFF状態でBを3秒長押し(赤点滅)。Cボタンで選択(赤=1、青=2、白=デフォルト)。B長押しで決定。
  • メモリー消去： Bを3秒長押し。Cで消去するメモリーを選択し、AとBを同時長押しして消去。

3. カラーチューニング機能 (アプリ連携)

  • 接続： スマホのBluetoothと位置情報をON。ペンライトをONにし、AとBを同時長押し（青ゆっくり点滅）。アプリで「Pairing」をタップし「KBX5」を選択。
  • アプリ設定： 「+Create Blade」でリスト作成、「+Create Color」で色追加。RGBWを調整して「Install」をタップし、メモリーAまたはBを選択。

4. ホットボタン機能 (推し色ジャンプ)

  • 設定： 推し色を点灯させた状態でCボタンを約3秒長押し（1回点滅で完了）。
  • 使用： どの色からでもCボタンを押すと推し色にジャンプ。
  • 戻る： 推し色点灯中にAまたはBボタンを押すと元の色に戻る。
